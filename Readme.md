# 🌐 Internship Web Project

This is a responsive static website built as part of my internship training. The project is developed using HTML5, CSS3, and Bootstrap framework for layout and styling.

## 📌 Project Overview

The goal of this project was to understand and implement:
- Responsive web design using Bootstrap
- Mobile-first layout development
- Reusable CSS components
- Navigation bar, sections, and footer layout

## 🛠️ Tools & Technologies
- HTML5
- CSS3
- Bootstrap 5

## ✨ Features
- Clean and responsive layout
- Bootstrap grid and components
- Structured sections (Hero, Services, Contact, etc.)
- Mobile-optimized navbar and footer
